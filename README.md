# wlog (Work Log)

### Install

```bash
go install github.com/almahoozi/wlog@latest
```

or you can install directly from the source:

```bash
make install
```

### Usage

Start the application, you will be greeted with all the hints you need to get started.

```bash
wlog
```

Write the list view to stdout:

```bash
wlog cat
wlog cat "last 3 days"
```

### Config

Run the TUI to see the defaults and modify config as needed.

```bash
wlog config
```
